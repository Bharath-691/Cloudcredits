# 🚀 CloudCredits AI & ML Internship Projects

This repository contains the projects completed as part of the **CloudCredits AI & ML Internship Program**.

The projects demonstrate practical applications of **Machine Learning, Deep Learning, Natural Language Processing (NLP), and Computer Vision** using Python, TensorFlow, and Streamlit.

---

## 📌 Projects Included

### 🧠 Task 1 – Handwritten Digit Recognition using CNN

A Computer Vision application that recognizes handwritten digits (0–9) using a Convolutional Neural Network (CNN) trained on the MNIST dataset.

#### Features

* Upload handwritten digit images
* Draw digits on an interactive canvas
* Camera capture support
* Real-time digit prediction
* Confidence score display
* Top-3 prediction probabilities
* Streamlit-based user interface

#### Technologies Used

* Python
* TensorFlow / Keras
* OpenCV
* NumPy
* Streamlit

---

### 🎬 Task 2 – Movie Review Sentiment Analysis

A Natural Language Processing (NLP) application that classifies movie reviews as **Positive** or **Negative** using machine learning techniques.

#### Features

* User review input
* Real-time sentiment prediction
* Confidence score display
* Interactive Streamlit interface
* Text preprocessing pipeline

#### Technologies Used

* Python
* Scikit-learn
* Pandas
* NumPy
* Streamlit
* NLP Techniques
